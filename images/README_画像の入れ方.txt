■ GitHub（GitHub Pages）で公開するときの注意

写真が表示されない場合は、ほぼ「画像ファイルがアップロードされていない」
「フォルダの場所がずれている」のどちらかです。次を確認してください。

1. index.html と images フォルダを、同じ階層にアップロードする
     （例）  リポジトリ/
              ├─ index.html
              └─ images/
                   ├─ logo.png
                   ├─ hero.jpg
                   ├─ board-ceiling.jpg
                   ├─ lgs-framing.jpg
                   └─ representative.jpg

2. images フォルダの中の画像5点が、すべてアップロードされているか確認する
     （1ファイルずつではなく、imagesフォルダごとアップロードするのが確実です）

3. ファイル名は大文字・小文字も区別されます。上記のとおり半角小文字で。

※ 公開URLで画像が出ない＝多くの場合この3点のいずれかです。
※ 万一読み込めなくても、ページは崩れず代替表示されるようにしています。
